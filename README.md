# Taller de Angular

Código fuente utilizado en el Taller de AngularJS impartido en #OpenExpoDay2015

## Instalación

Tener NodeJS instalado para poder descargar y utilizar las dependencias.
```
$ npm install
```

## Ejecución

Insertar el siguiente comando en la terminal
```
$ npm start
```

Abrir el navegador en la url: `http://localhost:8080`
